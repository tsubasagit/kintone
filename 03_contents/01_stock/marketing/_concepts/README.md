# 01_stock ディレクトリの定義

このディレクトリは、**AppTalentHub株式会社のオウンドメディア（[コラム・活動報告](https://apptalenthub.co.jp/category/column-posts/)）**に掲載するためのストック型コンテンツを管理する場所です。

## コンテンツの対象
- **技術的考察**: 開発エンジニアとしての知見、最新技術のレビュー、アーキテクチャの解説など。
- **会社の活動**: イベント登壇、ピッチコンテストの報告、社内プロジェクトの裏側など。
- **一般ビジネス/知見**: 業界動向の分析、開発プロセス、チームマネジメントなど。

## ディレクトリ構造の推奨案
各フォルダは一つの「記事」または「テーマ」として管理します。

```text
01_stock/
├── _concepts/           # 運用ルールや記事テンプレートの管理
├── 技術考察/            # テクノロジー関連の記事
│   └── architecture-2026/
├── 活動報告/            # イベントやピッチの記録
│   └── 伊東市アイディアピッチ/
└── 下書き/              # アイディア段階のもの
```

## 運用ワークフロー
1. **Stock**: このディレクトリ（`01_stock`）で原稿や素材を作成・構成。
2. **Review**: 内容の精査と校正。
3. **Published**: 記事として完成し、Webサイトにアップロードされたものは `02_published` へ移動（またはステータス変更）。
